# Locandina · 10 Agriturismi sull'Appennino Bolognese

Locandina HTML self-contained con 10 proposte per **pranzo + pomeriggio in altura** sopra i 500 m, da Casalecchio/Pianoro verso Monghidoro, Monterenzio, Castiglione dei Pepoli e il Mugello.

## 🌐 Online

Vedi la versione pubblicata su GitHub Pages:

> **https://elia-berselli.github.io/locandina-agriturismi-appennino/**

## 📂 Contenuto del repo

- `locandina.html` — la locandina (apribile direttamente nel browser, anche offline)
- `assets/` — 30 foto JPG delle 10 location (3 per ciascuna)
- `.nojekyll` — disabilita Jekyll su GitHub Pages (necessario)

## 🖱️ Funzionalità

- **Card cliccabili**: ogni location ha bottoni per Sito ufficiale, Facebook, Mappa Google Maps, TripAdvisor
- **Tabella riassuntiva** per confronto rapido
- **Sezione "Come condividere"** con 3 metodi pratici
- **Stampa PDF** nativa (`Ctrl+P` → "Salva come PDF")

## 🛠️ Tech

- HTML + CSS puri, nessuna build
- Font: Fraunces (serif titoli) + Inter (sans corpo) da Google Fonts
- Palette: rustico-moderno appennino (beige, marrone, verde salvia, senape)
- Responsive: layout 1200 px ottimizzato per desktop e stampa A3
